# Employee Hierarchy Using Inheritance

## Description
This Java program demonstrates the concept of inheritance and method overriding.
An Employee base class is created, and a Manager class extends Employee by adding
a department field and overriding the displayInfo() method.

## Objectives
- To understand inheritance in Java
- To demonstrate method overriding using super keyword

## Program Details
- Employee class contains name, id, and basicSalary
- Manager class extends Employee and adds department
- displayInfo() method is overridden in Manager class

## How to Run
1. Open the project in VS Code
2. Compile the program using:
   javac Report1.java
3. Run the program using:
   java Report1

## Output
The program displays the details of an Employee and a Manager.
